📌 Overview

This project analyzes Blinkit's sales and outlet performance using Power BI. The interactive dashboard provides insights into product demand, sales distribution, and outlet performance to support data-driven decision making.

🛠 Tools Used

Power BI

Power Query

DAX

Excel / CSV Dataset

📊 Key Insights

Total sales and average sales performance

Sales distribution by product category

Comparison of outlet types, sizes, and locations

Identification of top-performing products
